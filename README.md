This project is a Python-based finance tracker tailored to students to manage their allowance, log expenses by category, and view summaries with helpful saving suggestions.  
Originally developed as a partner output.

Features include: 
- Adding custom expense categories
- Inputting expenses weekly or monthly (weekly is auto-converted to monthly)
- Tracking total, daily, and weekly spending
- Calculating net gain/deficit vs. a goal budget
- Generating summaries with expense breakdowns, top 3 biggest expenses, and saving tips

How to run the program:
1. Clone this repository:
   ```bash
   git clone https://github.com/niki623/College-students-finance-tracker.git
   cd College-students-finance-tracker
2. Run the program:
   python tracker.py
